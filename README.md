# gpo-appflowy

Self-hosted [AppFlowy](https://github.com/AppFlowy-IO/AppFlowy-Cloud) on GKE, built from AGPL-3.0 source to run without the closed commercial images and their artificial user limit.

This is a monorepo of vendored upstream source (`upstream/`), Kubernetes manifests (`kubernetes/`), and the CI/CD that builds images and drives GitOps deploys (`.github/workflows/`).
